# lais-encuentro-2020

Proyecto de la página web del Encuentro Internacional REDOC Investigación para y sobre documentales

## Dependencias

[NodeJS](https://nodejs.org) (version LTS)

## Ambiente de desarrollo
```sh
npm install
npm run serve
```

### Compilar para desarrollo
```sh
npm run build
```
